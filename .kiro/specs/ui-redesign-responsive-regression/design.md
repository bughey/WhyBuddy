## 测试策略

### 自动化验证

1. **构建验证**：`pnpm run build` 成功
2. **类型检查**：`node --run check` 不引入新错误
3. **测试套件**：`pnpm run test` 不引入新失败
4. **暖色值扫描**：使用 grep 扫描残留暖色硬编码值

### 手动验证

1. **桌面端**：在 1280px、1440px、1728px 下检查布局
2. **平板端**：在 1024px、768px 下检查布局
3. **移动端**：在 375px、414px 下检查布局
4. **侧边栏交互**：折叠/展开过渡平滑
5. **发起面板**：打开/关闭动画正常
6. **键盘导航**：Tab 键在侧边栏、卡片、面板中正常工作
